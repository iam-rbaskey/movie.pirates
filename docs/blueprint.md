# **App Name**: CineRate

## Core Features:

- Landing Page UI: Display a clean, modern landing page with trending movie details and a search bar.
- Movie Detail Page: Provide a detailed movie information page including title, poster, cast, and user reviews.
- User Dashboard: Enable users to view and manage their profiles. 
- Review and Rating Submission: Allow users to submit star ratings and reviews for movies.
- User Authentication: Secure user authentication (register/login/logout).
- Theme Toggle: Dark/light mode toggle for UI.
- Movie Recommendation: Use an AI tool to suggest movies the user might enjoy based on rating histories.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke a sense of trust and reliability.
- Background color: Light gray (#F0F2F5), providing a neutral backdrop to make content pop.
- Accent color: Yellow-orange (#FFB347) to highlight interactive elements and call-to-actions, adding warmth and a playful touch.
- Body text: 'Inter', a grotesque sans-serif, will give a modern and machined look
- Headline Font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel; to make for techy, modern look
- Use a set of consistent, minimalist icons, mainly line icons, to ensure a clean and modern aesthetic.
- Subtle fade-in animations on content load to create a smooth user experience.
